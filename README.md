# Vijaya Xerox and Stationery E-Commerce System

A domain-specific e-commerce platform for a local bookstore and stationery business, specializing in academic and medical books.

## Architecture

This is a monorepo containing:

- **Backend API** (Next.js) - `/apps/api`
- **Customer App** (Flutter) - `/apps/customer_app` (planned)
- **Admin App** (Flutter) - `/apps/admin_app` (planned)

## Tech Stack

- **Backend**: Next.js 14, TypeScript, Prisma ORM
- **Database**: PostgreSQL 15 (External)
- **Cache**: Upstash Redis (Serverless)
- **Auth**: JWT tokens
- **Payment**: Razorpay (planned)
